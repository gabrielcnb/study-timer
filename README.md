# Study Timer Pomodoro

Desktop study timer with Pomodoro support, per-subject tracking, usage history and charts. Runs in the Windows system tray.

## Features

- Pomodoro, continuous and manual timer modes
- Track time per subject/category
- Session history exported to CSV
- Charts with daily/weekly statistics
- System tray with quick start/stop
- Custom icon and dark theme (customtkinter)

## Stack

- Python 3.10+
- customtkinter (modern UI)
- matplotlib (statistics charts)
- pystray (system tray)
- Pillow (icon handling)

## Usage

```bash
pip install customtkinter matplotlib pystray pillow
python main.py
```
